🌦️ Weather Forecast App

An interactive weather forecast web application built using React and Vite. This app allows users to search for any city and view real-time weather details including temperature, humidity, wind speed, and location coordinates.

🔗 Live Demo
👉 https://harshitha-965.github.io/Weather-Forecast/

✨ Features

🌍 Search weather by city name
🌡️ Displays temperature in Celsius
💧 Shows humidity levels
🌬️ Displays wind speed
📍 Shows latitude and longitude
❌ Handles invalid city input ("City not found")
⚡ Fast and responsive UI

🛠️ Technologies Used

- React.js
- Vite
- HTML5, CSS3, JavaScript
- OpenWeatherMap API

▶️ How to Run Locally

1. Clone the repository
git clone https://github.com/Harshitha-965/Weather-Forecast.git

2. Navigate into the project folder
cd "Weather app"

3. Install dependencies
npm install

4. Start the development server
npm run dev

5. Open in browser
http://localhost:5173/
